# sorting-visualizer
A web-based sorting algorithm visualizer that animates how different sorting algorithms work in real time.

## Algorithms Included
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

## Features
- Adjustable array size (6 to 40 elements)
- Speed control (Slow / Normal / Fast)
- Live stats — comparisons, swaps, and time elapsed
- Color coded bars — comparing, swapping, sorted, pivot

## How to Run
Just open `index.html` in any browser. No installation needed.
